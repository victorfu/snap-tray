## แท็บการตั้งค่า

แท็บ OCR และการบันทึกใช้ได้เฉพาะบน macOS และ Windows Linux beta ซ่อนทั้งสองแท็บเนื่องจากฟีเจอร์เหล่านั้นไม่รวมอยู่

### General

เปิดตอนสตาร์ทอัป ภาษาการแสดงผล ธีมแอป และการติดตั้ง/ถอนการติดตั้ง CLI

บน macOS แท็บนี้ยังแสดงสถานะสิทธิ์ Screen Recording และ Accessibility พร้อมลิงก์ตั้งค่าระบบแบบคลิกเดียว

### Hotkeys

ตัวแก้ไขปุ่มลัดแบบรวมศูนย์ตามหมวดหมู่การทำงาน

### Advanced

การสลับแว่นขยายการจับภาพเฉพาะส่วน คำแนะนำปุ่มลัดการจับภาพ ความเข้ม/ประเภทการเบลอ และค่าเริ่มต้นหน้าต่างปักหมุด (ความโปร่งใส ขั้นตอนความโปร่งใส ขั้นตอนการซูม ขีดจำกัดแคช)

### Watermark

การเลือกภาพลายน้ำ ว่าจะใช้กับสกรีนช็อต/การบันทึกหรือไม่ ความโปร่งใส ขนาด ระยะขอบ และตัวอย่างตำแหน่ง

### OCR

ลำดับภาษา OCR และพฤติกรรมหลังการรู้จำ

### Recording

เฟรมเรต รูปแบบเอาต์พุตเริ่มต้น พฤติกรรมการส่งออกแบบดูตัวอย่างก่อน แหล่งเสียง/อุปกรณ์ และการนับถอยหลัง

### Files

โฟลเดอร์สกรีนช็อตและการบันทึก รวมถึงเทมเพลตรูปแบบชื่อไฟล์

### Updates

เวอร์ชันปัจจุบัน ช่วงเวลาตรวจสอบอัตโนมัติ สถานะตรวจสอบล่าสุด และการตรวจสอบด้วยตนเองทันที

### About

ชื่อแอป เวอร์ชัน ลิขสิทธิ์ ผู้เขียน และลิงก์เว็บไซต์

## โปรไฟล์พื้นฐานที่แนะนำ

- บน macOS/Windows ใช้ MP4 ที่ 30 FPS สำหรับบทช่วยสอนมาตรฐาน
- โฟลเดอร์เฉพาะสำหรับสกรีนช็อต และการบันทึกในกรณีที่ใช้ได้
- เปิดการตรวจสอบอัปเดตอัตโนมัติรายวัน
