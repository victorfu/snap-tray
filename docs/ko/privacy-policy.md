## 개요

SnapTray는 로컬 우선의 스크린샷·주석 앱입니다. 녹화 기능은 macOS와 Windows에서만 지원됩니다. Linux beta는 스크린샷과 주석만 지원합니다. 네트워크 연결은 공유 URL 업로드를 직접 사용할 때 또는 선택적 업데이트 확인이 실행될 때만 이루어집니다. 저희는 개인정보 보호와 투명한 운영을 중요하게 생각합니다.

**핵심 요약:** SnapTray는 로컬 우선으로 동작하며 원격 분석·추적·분석 기능이 일체 없습니다. 네트워크 통신은 공유 URL(사용자가 직접 실행하는 이미지 업로드)과 업데이트 확인(버전 문자열만 전송) 두 가지 명시적 기능에만 한정됩니다.

## 데이터 수집

**어떠한 데이터도 수집하지 않습니다.** 구체적으로:

- 원격 분석 또는 사용 통계 분석을 수행하지 않습니다
- 외부 서버에 충돌 보고서를 전송하지 않습니다
- 사용자를 추적하거나 고유 식별자를 생성하지 않습니다

## 데이터 저장

SnapTray가 사용하거나 생성하는 모든 데이터는 사용자 기기에 로컬로 저장됩니다:

| 데이터 유형 | 저장 위치 | 용도 |
|------------|----------|------|
| 스크린샷 | 사용자 지정 폴더(기본값: 사진) | 캡처한 이미지 저장 |
| 녹화 | 사용자 지정 폴더(기본값: 다운로드) | macOS와 Windows 화면 녹화 저장. Linux beta는 녹화 기능 미포함 |
| 설정 | OS 설정 저장소* | 환경설정 저장 |

*설정 저장 위치:
- **macOS:** `~/Library/Preferences/cc.snaptray.plist`
- **Windows:** `HKEY_CURRENT_USER\Software\SnapTray`
- **Linux beta:** Qt/Linux 사용자 설정 저장소(일반적으로 `~/.config/<organization>/<application>.conf`)
- 새 버전으로 업그레이드 후 첫 실행 시 이전 버전 설정이 자동으로 마이그레이션되며, 마이그레이션 성공 후 이전 데이터는 삭제됩니다.

## 권한

SnapTray는 기능에 따라 시스템 권한을 요청합니다:

### macOS

| 권한 | 용도 |
|------|------|
| Screen Recording | 스크린샷 및 녹화 핵심 기능 |
| Accessibility | 스마트 캡처를 위한 창 감지 |

### Windows

| 권한 | 용도 |
|------|------|
| Full Trust | DXGI 화면 캡처, WASAPI 오디오 캡처, OCR 기능에 필요 |

**선택적 권한:**
- 마이크 접근은 오디오 녹화를 선택한 경우에만 요청됩니다.

## 네트워크 통신

SnapTray가 연결하는 경우는 다음 두 가지 명시적 기능뿐입니다:

### 공유 URL

공유 URL 기능을 직접 사용하면 캡처한 이미지가 `x.snaptray.cc`에 업로드됩니다. 비밀번호를 선택적으로 설정할 수 있습니다. 공유 링크는 자동으로 만료됩니다. 사용자가 직접 실행하지 않는 한 업로드는 이루어지지 않습니다.

### 업데이트 확인

SnapTray는 새 릴리스를 `api.github.com`에서 확인합니다. 이 기능은 기본으로 활성화되어 있으며 설정에서 비활성화할 수 있습니다. 앱 버전 문자열만 전송됩니다. 개인 정보, 사용 데이터, 기기 식별자는 일체 전송되지 않습니다. 스토어 설치(Microsoft Store, Mac App Store 등)의 경우 업데이트 확인을 건너뜁니다.

### SnapTray가 하지 않는 것

- 추적 SDK 또는 분석 도구를 포함하지 않습니다
- 쿠키 또는 유사 기술을 사용하지 않습니다
- 원격 분석 또는 충돌 보고서를 전송하지 않습니다
