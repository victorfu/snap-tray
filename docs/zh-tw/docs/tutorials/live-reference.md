## 目標

把重要參考圖固定在前景，工作時不用一直切換視窗。

## 適用情境

- 依照設計稿做 UI 實作
- 對照預期與實際行為
- 測試時固定顯示驗收條件

## 操作流程

### 1. 擷取並釘選參考圖

1. 按 `F2` 擷取參考區域。
2. 按 `Enter` 將結果釘選到畫面。

### 2. 調整釘選視窗

- 調整大小，只保留必要資訊。
- 移動到螢幕邊角避免遮擋。
- 調整透明度，兼顧可見與不干擾。

### 3. 多釘選分工

複雜任務可分三個釘選：

- 設計目標圖
- 目前實作畫面
- QA 驗收條件

### 4. 使用歷史快速回復

當上下文常切換時，從 pin history 回復先前參考圖，比重截更快。

## 實務建議

- 一張小 pin 放文字需求，一張大 pin 放視覺細節。
- 若你啟用了 Quick Pin（預設 `Shift+F2`），可加速釘選流程。
- 遇到遮擋先降透明度，再調整位置。

## 品質檢查清單

- 參考內容在整段工作中都可見。
- alt-tab 次數明顯下降。
- 可從歷史快速還原重要參考圖。
