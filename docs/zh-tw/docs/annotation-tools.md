## 工具分類

### 繪圖

- Pencil
- Marker
- Arrow（直線與折線模式）
- Shape（矩形／橢圓）

### 說明標記

- Text
- Step Badge
- Emoji Sticker

### 隱私與修正

- Mosaic
- Eraser

## 建議編修流程

1. 先完成主體標線。
2. 統一顏色與線寬。
3. 補上文字與步驟編號。
4. 對敏感區域套用馬賽克。
5. 透過 Undo/Redo 微調後輸出。

## 使用建議

- 關鍵訊息維持單一強調色。
- Marker 適合高亮，不適合大量文字。
- 操作教學建議優先使用步驟標記。

## 匯出動作

- `Enter` 釘選
- `Ctrl/Cmd + C` 複製
- `Ctrl/Cmd + S` 儲存
