## Settings tabs

### General

Launch at startup, toolbar style, pin cache behavior, and CLI install/uninstall.

### Hotkeys

Centralized hotkey editor by functional category.

### Recording

Frame rate, format, quality, countdown, preview, and audio source.

### Files

Screenshot and recording folders plus filename format templates.

### OCR / Watermark / Updates

Language preference, watermark placement, auto-check update interval.

## Recommended baseline profile

- 30 FPS MP4 for standard tutorials
- Dedicated folders for screenshots and recordings
- Auto update checks enabled weekly
