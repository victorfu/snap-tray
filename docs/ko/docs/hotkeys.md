## 기본 단축키

| 동작 | 기본값 |
|---|---|
| 영역 캡처 | `F2` |
| 스크린 캔버스 | `Ctrl+F2 / Cmd+F2` |
| 붙여넣기 | `F3` |
| 빠른 고정 | `Shift+F2` |

## 추가 설정 가능 동작 (기본값 없음)

- 이미지로 고정
- 핀 기록
- 모든 핀 숨기기/표시 전환
- 화면 녹화 (macOS/Windows 전용)

## 단축키 카테고리

단축키는 설정 내에서 카테고리별로 관리됩니다.

- 캡처
- 캔버스
- 클립보드
- 핀
- 녹화 (macOS/Windows 전용; Linux beta에서는 숨김)

## 충돌 해결 전략

단축키가 다른 앱과 충돌하는 경우:

1. 설정 > 단축키에서 동작을 다시 바인딩합니다.
2. 즉시 트리거를 테스트합니다.
3. 자주 사용하는 동작은 단순한 단일 키 조합으로 유지합니다.

## 팀 권장 사항

공유 환경에서는 표준 단축키 프로필을 정의하고 내부적으로 문서화하세요.
