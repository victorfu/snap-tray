## ツールカテゴリ

### 描画

- ペン
- マーカー
- 矢印（単線とポリライン モード）
- 図形（長方形/楕円）

### マークアップとガイダンス

- テキスト
- ステップバッジ
- 絵文字スタンプ

### プライバシーと修正

- モザイク
- 消しゴム

## 編集ワークフロー

1. まず大まかな注釈を描きます。
2. 色と幅をグローバルに調整します。
3. テキストとステップバッジを追加して手順を示します。
4. 機密部分にモザイクを適用します。
5. エクスポート前に元に戻す/やり直しを使います。

## ベストプラクティス

- 重要な箇所には 1 色のアクセントカラーを使います。
- マーカーは強調に使い、密なテキストには使わないようにしましょう。
- 手順のスクリーンショットにはステップバッジを優先して使います。

## エクスポートアクション

- `Enter` で結果をピン留め
- `Ctrl/Cmd + C` でコピー
- `Ctrl/Cmd + S` で保存
