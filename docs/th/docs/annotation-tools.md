## ประเภทของเครื่องมือ

### การวาด

- ดินสอ
- มาร์กเกอร์
- ลูกศร (โหมดเส้นเดียวและโพลีไลน์)
- รูปทรง (สี่เหลี่ยม/วงรี)

### มาร์กอัปและคำแนะนำ

- ข้อความ
- แบดจ์ขั้นตอน
- สติกเกอร์อีโมจิ

### ความเป็นส่วนตัวและการแก้ไข

- โมเสก
- ยางลบ

## เวิร์กโฟลว์การแก้ไข

1. วาดมาร์กอัปคร่าวๆ ก่อน
2. ปรับสีและความกว้างโดยรวม
3. เพิ่มข้อความและแบดจ์ขั้นตอนสำหรับลำดับ
4. ใช้โมเสกในพื้นที่ที่มีข้อมูลสำคัญ
5. ใช้ Undo/Redo ก่อนส่งออก

## แนวทางปฏิบัติที่ดี

- ใช้สีเน้นสีเดียวสำหรับจุดสำคัญ
- ใช้มาร์กเกอร์เพื่อเน้นย้ำ ไม่ใช่สำหรับข้อความหนาแน่น
- ควรใช้แบดจ์ขั้นตอนสำหรับสกรีนช็อตที่มีขั้นตอน

## การดำเนินการส่งออก

- `Enter` เพื่อปักหมุดผลลัพธ์
- `Ctrl/Cmd + C` เพื่อคัดลอก
- `Ctrl/Cmd + S` เพื่อบันทึก

## ขั้นตอนถัดไป

หากต้องการมาร์กอัปโดยตรงบนเดสก์ท็อป เปิด [สกรีนแคนวาส](/th/docs/screen-canvas/)
