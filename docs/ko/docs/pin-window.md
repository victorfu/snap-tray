## 핀 창 만들기

캡처 결과에서 `Enter`를 누르거나, 트레이 메뉴의 "이미지로 고정..."을 사용하여 고정할 수 있습니다.

## 핵심 컨트롤

### 마우스 동작

- 드래그하여 이동
- 마우스 휠로 확대/축소
- `Ctrl` + 휠로 불투명도 조정
- 가장자리를 드래그하여 크기 조정

### 키보드 동작

| 동작 | 키 |
|---|---|
| 시계 방향 회전 | `1` |
| 반시계 방향 회전 | `2` |
| 수평 뒤집기 | `3` |
| 수직 뒤집기 | `4` |
| 닫기 | `Esc` |

## 컨텍스트 메뉴 동작

우클릭 메뉴에서 다음 동작을 사용할 수 있습니다.

- 보기/레이아웃: 툴바 표시, 테두리 표시, 화면으로 이동, 핀 병합, 영역 레이아웃 조정
- 이미지 컨트롤: 확대/축소 프리셋, 이미지 처리 (회전/뒤집기/자르기/불투명도 단계), 스무딩
- 출력/워크플로: 복사, 저장, 캐시 폴더 열기, 꾸미기, 워터마크, 실시간 업데이트
- 상호작용/닫기: 클릭 통과, 닫기, 모든 핀 닫기

## 핀 창에서 주석

연필 아이콘을 클릭하여 현재 고정된 이미지 전용 경량 주석 툴바를 엽니다.

- 그리기 도구: 연필, 마커, 화살표, 도형, 텍스트, 모자이크, 단계 배지, 이모지, 지우개
- 유틸리티 도구: 자르기, 측정
- 동작 도구: OCR (macOS/Windows 전용; Linux beta에서는 숨김), QR 코드 스캔, URL 공유, 저장, 복사
