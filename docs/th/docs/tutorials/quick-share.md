## เป้าหมาย

สร้างสกรีนช็อตที่สื่อสารประเด็นสำคัญหนึ่งอย่างและแชร์ทันที

## เวลาและการตั้งค่า

- เวลา: 30-60 วินาที
- ปุ่มลัดเริ่มต้น: `F2`
- เอาต์พุต: คลิปบอร์ด (เร็วที่สุด) หรือไฟล์ (ติดตามได้)

## เวิร์กโฟลว์

### 1. จับภาพพื้นที่ที่ต้องการ

1. กด `F2`
2. ลากเพื่อเลือกส่วนเป้าหมาย
3. ใช้แว่นขยายและ overlay ขนาดเพื่อความแม่นยำ

เคล็ดลับ: หากหน้าต่างแอปมีขอบที่ชัดเจน ให้คลิกหน้าต่างที่ตรวจพบโดยตรงแทนการลากด้วยตนเอง

### 2. เพิ่มมาร์กอัปที่เน้นจุดเดียว

1. เลือก `Arrow` หรือ `Marker`
2. เน้นเฉพาะพื้นที่สำคัญ
3. หากต้องการข้อความ เพิ่มป้ายกำกับสั้นๆ ด้วย `Text`

กฎ: ควรใช้มาร์กอัปที่แข็งแกร่งอันเดียวดีกว่าเครื่องหมายอ่อนหลายอัน

### 3. ลบข้อมูลสำคัญ

1. เลือก `Mosaic`
2. ปิดบังอีเมล รหัส หรือชื่อส่วนตัว

### 4. แชร์ทันที

- `Ctrl/Cmd + C`: คัดลอกไปยังคลิปบอร์ด
- วางลงใน chat/อีเมล/ticket
- `Share URL`: สร้างลิงก์แชร์ (รหัสผ่านเสริม) เมื่อต้องการแชร์ภายนอก
- เสริม: `Ctrl/Cmd + S` สำหรับบันทึกไฟล์

## รายการตรวจสอบคุณภาพ

- พื้นที่เป้าหมายถูกตัดอย่างแน่นหนา
- ประเด็นสำคัญหนึ่งอย่างถูกเน้น
- ข้อมูลสำคัญถูกซ่อน
- ผู้รับสามารถเข้าใจบริบทได้โดยไม่ต้องอธิบายเพิ่มเติม
