## ภาพรวม

SnapTray เป็นแอปสกรีนช็อตและการมาร์กอัปที่เน้นการทำงานในเครื่อง โดยการบันทึกหน้าจอรองรับเฉพาะ macOS และ Windows เท่านั้น Linux beta รองรับเฉพาะการสกรีนช็อตและการมาร์กอัป การเชื่อมต่อเครือข่ายเกิดขึ้นเฉพาะเมื่อคุณใช้ฟีเจอร์แชร์ URL อย่างตั้งใจ หรือเมื่อมีการตรวจสอบอัปเดตแบบเลือกได้ เราให้ความสำคัญกับความเป็นส่วนตัวและดำเนินการอย่างโปร่งใส

**สรุปสำคัญ:** SnapTray ทำงานโดยเน้นข้อมูลในเครื่อง ไม่มีการวิเคราะห์ ติดตาม หรือส่งข้อมูลใดๆ การเชื่อมต่อเครือข่ายจำกัดอยู่เพียงสองฟีเจอร์ที่ชัดเจน ได้แก่ แชร์ URL (การอัปโหลดรูปภาพที่ผู้ใช้เริ่มต้นเอง) และการตรวจสอบอัปเดตแบบเลือกได้ (ส่งเฉพาะหมายเลขเวอร์ชันเท่านั้น)

## การเก็บข้อมูล

**เราไม่เก็บข้อมูลใดทั้งสิ้น** ซึ่งรวมถึง:

- ไม่มีการวิเคราะห์พฤติกรรมการใช้งานหรือการส่งข้อมูลทางไกล
- ไม่มีการส่งรายงานการขัดข้องไปยังเซิร์ฟเวอร์ภายนอก
- ไม่มีการติดตามหรือสร้างตัวระบุที่ไม่ซ้ำกัน

## การจัดเก็บข้อมูล

ข้อมูลทั้งหมดที่ SnapTray ใช้หรือสร้างขึ้นจะถูกเก็บในเครื่องของคุณ:

| ประเภทข้อมูล | สถานที่จัดเก็บ | วัตถุประสงค์ |
|--------------|----------------|--------------|
| สกรีนช็อต | โฟลเดอร์ที่ผู้ใช้กำหนด (ค่าเริ่มต้น: รูปภาพ) | จัดเก็บภาพที่จับ |
| การบันทึกหน้าจอ | โฟลเดอร์ที่ผู้ใช้กำหนด (ค่าเริ่มต้น: ดาวน์โหลด) | จัดเก็บการบันทึกบน macOS และ Windows; Linux beta ไม่รวมฟีเจอร์การบันทึก |
| การตั้งค่า | พื้นที่จัดเก็บการตั้งค่าของระบบปฏิบัติการ* | จัดเก็บการตั้งค่าต่างๆ |

*สถานที่จัดเก็บการตั้งค่า:
- **macOS:** `~/Library/Preferences/cc.snaptray.plist`
- **Windows:** `HKEY_CURRENT_USER\Software\SnapTray`
- **Linux beta:** พื้นที่จัดเก็บการตั้งค่าผู้ใช้ Qt/Linux มักอยู่ที่ `~/.config/<organization>/<application>.conf`
- เมื่ออัปเกรดเป็นเวอร์ชันใหม่ การตั้งค่าของเวอร์ชันเก่าจะถูกย้ายโดยอัตโนมัติในการเปิดใช้งานครั้งแรก และข้อมูลเก่าจะถูกลบหลังจากย้ายสำเร็จ

## สิทธิ์ที่จำเป็น

SnapTray ร้องขอสิทธิ์ของระบบตามฟีเจอร์ที่ใช้:

### macOS

| สิทธิ์ | วัตถุประสงค์ |
|--------|--------------|
| Screen Recording | ฟีเจอร์หลักสำหรับสกรีนช็อตและการบันทึกหน้าจอ |
| Accessibility | การตรวจจับหน้าต่างสำหรับฟีเจอร์การจับภาพขั้นสูง |

### Windows

| สิทธิ์ | วัตถุประสงค์ |
|--------|--------------|
| Full Trust | จำเป็นสำหรับการจับภาพ DXGI การบันทึกเสียง WASAPI และความสามารถ OCR |

**สิทธิ์เสริม:**
- การเข้าถึงไมโครโฟนจะถูกร้องขอเฉพาะเมื่อคุณเลือกบันทึกพร้อมเสียง

## การสื่อสารทางเครือข่าย

SnapTray เชื่อมต่อเครือข่ายเฉพาะสองฟีเจอร์ที่ชัดเจนเท่านั้น:

### แชร์ URL

เมื่อคุณใช้ฟีเจอร์แชร์ URL อย่างตั้งใจ ภาพที่จับจะถูกอัปโหลดไปยัง `x.snaptray.cc` คุณสามารถเลือกตั้งรหัสผ่านป้องกันได้ ลิงก์แชร์จะหมดอายุโดยอัตโนมัติ ไม่มีการอัปโหลดใดๆ หากคุณไม่ได้เริ่มต้นการดำเนินการ

### การตรวจสอบอัปเดต

SnapTray ตรวจสอบเวอร์ชันใหม่จาก `api.github.com` ฟีเจอร์นี้เปิดใช้งานโดยค่าเริ่มต้นและสามารถปิดได้ในการตั้งค่า ข้อมูลที่ส่งมีเพียงหมายเลขเวอร์ชันของแอปเท่านั้น ไม่มีข้อมูลส่วนตัว ข้อมูลการใช้งาน หรือตัวระบุอุปกรณ์ การติดตั้งจากสโตร์ (เช่น Microsoft Store หรือ Mac App Store) จะข้ามการตรวจสอบนี้

### สิ่งที่ SnapTray ไม่ทำ

- ไม่รวม SDK การติดตามหรือเครื่องมือวิเคราะห์
- ไม่ใช้คุกกี้หรือเทคโนโลยีที่คล้ายกัน
- ไม่ส่งข้อมูลทางไกลหรือรายงานการขัดข้อง
