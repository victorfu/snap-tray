## Enter canvas mode

Press `Ctrl+F2` (default) or use tray menu "Screen Canvas".

## Available tools

- Pencil / Marker / Arrow / Shape
- Text / Step Badge / Emoji
- Laser Pointer for presentation

## Background modes

### Whiteboard

Best for dark room projection or clean note visuals.

### Blackboard

Better contrast when highlighting bright colors.

## Session controls

- Undo / Redo
- Clear canvas
- `Esc` to exit

## When to use canvas vs capture

Use Region Capture for static snapshots and exports. Use Screen Canvas for live explanation on top of current desktop context.
