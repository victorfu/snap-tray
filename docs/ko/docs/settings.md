## 설정 탭

OCR 및 녹화 탭은 macOS 및 Windows에서만 사용 가능합니다. Linux beta는
해당 기능이 포함되지 않아 두 탭 모두 숨겨집니다.

### 일반

시작 시 실행, 표시 언어, 앱 테마, CLI 설치/제거.

macOS에서는 이 탭에 화면 녹화 및 접근성 권한 상태와 원클릭 시스템 설정 바로가기도 표시됩니다.

### 단축키

기능 카테고리별 중앙 집중식 단축키 편집기.

### 고급

영역 캡처 돋보기 전환, 캡처 단축키 힌트, 블러 강도/유형, 핀 창 기본값 (불투명도, 불투명도 단계, 확대/축소 단계, 캐시 제한).

### 워터마크

워터마크 이미지 선택, 스크린샷/녹화 적용 여부, 불투명도, 크기, 여백, 위치 미리보기.

### OCR

OCR 언어 순서 및 인식 후 동작.

### 녹화

프레임 레이트, 기본 출력 형식, 미리보기 후 내보내기 동작, 오디오 소스/장치, 카운트다운.

### 파일

스크린샷 및 녹화 폴더와 파일명 형식 템플릿.

### 업데이트

현재 버전, 자동 확인 간격, 마지막 확인 상태, 수동 지금 확인.

### 정보

앱 이름, 버전, 저작권, 제작자, 웹사이트 링크.

## 권장 기본 프로필

- macOS/Windows에서 표준 튜토리얼은 30 FPS MP4 사용
- 스크린샷과 (지원되는 플랫폼의) 녹화를 위한 전용 폴더 지정
- 매일 자동 업데이트 확인 활성화
