## キャンバスモードに入る

`Ctrl+F2 / Cmd+F2`（デフォルト）を押すか、トレイメニューの「Screen Canvas」を使います。

## 利用可能なツール

- ペン / マーカー / 矢印 / 図形
- テキスト / ステップバッジ / 絵文字
- プレゼンテーション用レーザーポインター

## 背景モード

### ホワイトボード

暗い部屋でのプロジェクションや、クリーンなノートビジュアルに最適です。

### ブラックボード

明るい色を強調する場合のコントラストが優れています。

## セッションコントロール

- 元に戻す / やり直し
- キャンバスをクリア
- `Esc` で終了

## キャンバスとキャプチャの使い分け

静的なスナップショットとエクスポートには範囲キャプチャを使います。現在のデスクトップコンテキスト上でのライブ説明にはスクリーンキャンバスを使います。
