## このセクションの使い方

SnapTray が初めての方は、まず **バグ報告ワークフロー** から始めてください。各チュートリアルはタスクベースで、目標、手順、最終確認チェックリストが含まれています。

## 推奨学習パス

| 目標 | まずここから | 次に続ける |
|---|---|---|
| 明確な証拠でバグを報告する | [バグ報告ワークフロー](/ja/docs/tutorials/bug-report/) | [注釈ツール](/ja/docs/annotation-tools/) |
| 作業中も参照を常時表示する | [ピン留めで常時参照](/ja/docs/tutorials/live-reference/) | [ピン留めウィンドウ](/ja/docs/pin-window/) |
| 製品デモを録画・共有する（macOS/Windows のみ） | [録画デモ（macOS/Windows のみ）](/ja/docs/tutorials/recording-demo/) | [録画（macOS/Windows のみ）](/ja/docs/recording/) |

## チュートリアルのスタイル

- すべてのガイドは繰り返し実行可能なワークフローとして書かれています。
- デフォルトのショートカットが含まれているので素早く実行できます。
- 各ガイドは品質チェックリストで終わり、出力が共有可能な状態であることを確認できます。

## 始める前に

1. [はじめに](/ja/docs/getting-started/) を完了してください。
2. [ホットキー](/ja/docs/hotkeys/) でホットキーを確認してください。
3. [設定](/ja/docs/settings/) で出力フォルダーを確認してください。
