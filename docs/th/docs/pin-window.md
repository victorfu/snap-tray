## สร้างหน้าต่างปักหมุด

คุณสามารถปักหมุดจากผลลัพธ์การจับภาพ (`Enter`) หรือจากเมนูถาดระบบโดยใช้ "Pin from Image..."

## การควบคุมหลัก

### การดำเนินการด้วยเมาส์

- ลากเพื่อย้าย
- เลื่อนล้อเมาส์เพื่อซูม
- `Ctrl` + เลื่อนล้อเพื่อปรับความโปร่งใส
- ลากขอบเพื่อปรับขนาด

### การดำเนินการด้วยแป้นพิมพ์

| การดำเนินการ | ปุ่ม |
|---|---|
| หมุนตามเข็มนาฬิกา | `1` |
| หมุนทวนเข็มนาฬิกา | `2` |
| พลิกแนวนอน | `3` |
| พลิกแนวตั้ง | `4` |
| ปิด | `Esc` |

## การดำเนินการในเมนูคลิกขวา

เมนูคลิกขวาประกอบด้วย:

- มุมมอง/จัดวาง: Show Toolbar, Show Border, Move to Screen, Merge Pins, Adjust Region Layout
- การควบคุมภาพ: ค่าซูมที่ตั้งไว้ล่วงหน้า การประมวลผลภาพ (หมุน/พลิก/ครอป/ขั้นตอนความโปร่งใส) ความเรียบ
- เอาต์พุต/เวิร์กโฟลว์: Copy, Save, Open Cache Folder, Beautify, Watermark, Live Update
- การโต้ตอบ/ปิด: Click-through, Close, Close All Pins

## การมาร์กอัปในหน้าต่างปักหมุด

ใช้ไอคอนดินสอเพื่อเปิดแถบเครื่องมือมาร์กอัปขนาดเล็กที่ออกแบบมาสำหรับภาพที่ปักหมุดอยู่

- เครื่องมือวาด: Pencil, Marker, Arrow, Shape, Text, Mosaic, Step Badge, Emoji, Eraser
- เครื่องมืออรรถประโยชน์: Crop, Measure
- เครื่องมือดำเนินการ: OCR (เฉพาะ macOS/Windows; ซ่อนใน Linux beta), QR Code Scan, Share URL, Save, Copy

## ขั้นตอนถัดไป

ใช้ [ปุ่มลัด](/th/docs/hotkeys/) เพื่อกำหนดการดำเนินการปักหมุดสำหรับเวิร์กโฟลว์ที่เร็วขึ้น
