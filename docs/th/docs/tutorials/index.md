## วิธีใช้ส่วนนี้

หากเพิ่งเริ่มใช้ SnapTray เริ่มด้วย **ขั้นตอนรายงานบัก** ก่อน บทช่วยสอนแต่ละบทอิงตามงานและมีเป้าหมาย ขั้นตอน และรายการตรวจสอบผลลัพธ์ในตอนท้าย

## เส้นทางการเรียนรู้ที่แนะนำ

| เป้าหมายของคุณ | เริ่มที่นี่ | แล้วดำเนินการต่อ |
|---|---|---|
| รายงานบักพร้อมหลักฐานชัดเจน | [ขั้นตอนรายงานบัก](/th/docs/tutorials/bug-report/) | [เครื่องมือมาร์กอัป](/th/docs/annotation-tools/) |
| เก็บอ้างอิงให้มองเห็นขณะทำงาน | [อ้างอิงสดด้วยการปักหมุด](/th/docs/tutorials/live-reference/) | [หน้าต่างปักหมุด](/th/docs/pin-window/) |
| บันทึกและแชร์สาธิตผลิตภัณฑ์ (เฉพาะ macOS/Windows) | [สาธิตการบันทึก (เฉพาะ macOS/Windows)](/th/docs/tutorials/recording-demo/) | [การบันทึกหน้าจอ (เฉพาะ macOS/Windows)](/th/docs/recording/) |

## รูปแบบบทช่วยสอน

- ทุกคู่มือเขียนเป็นเวิร์กโฟลว์ที่ทำซ้ำได้
- มีปุ่มลัดเริ่มต้นเพื่อให้ทำงานได้เร็ว
- ทุกคู่มือจบด้วยรายการตรวจสอบคุณภาพเพื่อให้เอาต์พุตพร้อมแชร์

## ก่อนเริ่ม

1. ทำ [เริ่มต้นใช้งาน](/th/docs/getting-started/) ให้เสร็จ
2. ยืนยันปุ่มลัดใน [ปุ่มลัด](/th/docs/hotkeys/)
3. ยืนยันโฟลเดอร์เอาต์พุตใน [การตั้งค่า](/th/docs/settings/)

## ต้องการรายละเอียดฟีเจอร์?

บทช่วยสอนมุ่งเน้นที่ผลลัพธ์ สำหรับข้อมูลอ้างอิงตัวเลือกทั้งหมด ใช้:

- [การจับภาพเฉพาะส่วน](/th/docs/region-capture/)
- [เครื่องมือมาร์กอัป](/th/docs/annotation-tools/)
- [หน้าต่างปักหมุด](/th/docs/pin-window/)
- [การบันทึกหน้าจอ (เฉพาะ macOS/Windows)](/th/docs/recording/)
- [CLI](/th/docs/cli/)
