녹화는 macOS 및 Windows에서만 사용 가능합니다. Linux beta는 녹화를 포함하지 않으며,
녹화 관련 UI가 숨겨져 있습니다.

## 녹화 진입점

- 트레이 메뉴: 화면 녹화
- 녹화 단축키: 설정 > 단축키에서 화면 녹화 구성

## 녹화 라이프사이클

1. 다중 디스플레이 설정에서 프롬프트가 표시될 때 녹화할 화면을 선택합니다.
2. 선택한 화면에서 즉시 녹화가 시작됩니다.
3. 플로팅 컨트롤 바를 사용하여 녹화 시간을 모니터링하고 중지합니다.
4. 중지를 클릭하여 내보냅니다.

## 출력 형식

| 형식 | 주요 용도 |
|---|---|
| MP4 (H.264) | 튜토리얼, 긴 녹화 |
| GIF | 짧은 반복 데모 |
| WebP | 경량 애니메이션 클립 |

## 오디오 옵션

MP4 녹화에서 현재 플랫폼과 소스가 지원하는 경우 오디오 캡처를 사용할 수 있습니다.

- 마이크
- 시스템 오디오
- 혼합 캡처

GIF와 WebP 내보내기는 소리가 없습니다.

## 품질 조정

설정 > 녹화를 열어 프레임 레이트, 품질, 카운트다운, 미리보기 동작을 조정하세요.
