## 목표

하나의 핵심 포인트를 전달하는 스크린샷을 만들어 즉시 공유합니다.

## 시간 및 설정

- 소요 시간: 30~60초
- 기본 단축키: `F2`
- 출력: 클립보드 (가장 빠름) 또는 파일 (추적 가능)

## 워크플로

### 1. 정확한 영역 캡처

1. `F2`를 누릅니다.
2. 드래그하여 대상 영역을 선택합니다.
3. 돋보기와 크기 오버레이를 활용하여 정밀하게 선택합니다.

팁: 앱 창에 명확한 테두리가 있다면 수동 드래그 대신 감지된 창을 직접 클릭하세요.

### 2. 초점이 맞춰진 주석 하나 추가

1. `화살표` 또는 `마커`를 선택합니다.
2. 핵심 영역만 강조합니다.
3. 텍스트가 필요하다면 `텍스트`로 짧은 레이블 하나를 추가합니다.

원칙: 많은 약한 마크보다 하나의 강한 주석을 선호하세요.

### 3. 민감한 정보 제거

1. `모자이크`를 선택합니다.
2. 이메일 주소, ID, 개인 이름을 가립니다.

### 4. 즉시 공유

- `Ctrl/Cmd + C`: 클립보드에 복사
- 채팅/이메일/티켓에 붙여넣기
- `URL 공유`: 외부 공유가 필요할 때 공유 링크 생성 (선택적 비밀번호)
- 선택 사항: `Ctrl/Cmd + S`로 파일 기록

## 품질 체크리스트

- 대상 영역이 타이트하게 잘려 있습니다.
- 핵심 포인트 하나가 강조되어 있습니다.
- 민감한 정보가 숨겨져 있습니다.
- 수신자가 추가 설명 없이도 맥락을 이해할 수 있습니다.
