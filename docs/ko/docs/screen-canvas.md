## 캔버스 모드 진입

`Ctrl+F2 / Cmd+F2`(기본값)를 누르거나 트레이 메뉴의 "스크린 캔버스"를 사용합니다.

## 사용 가능한 도구

- 연필 / 마커 / 화살표 / 도형
- 텍스트 / 단계 배지 / 이모지
- 프레젠테이션용 레이저 포인터

## 배경 모드

### 화이트보드

어두운 방에서의 프로젝션 또는 깔끔한 메모 화면에 적합합니다.

### 블랙보드

밝은 색상을 강조할 때 더 나은 대비를 제공합니다.

## 세션 컨트롤

- 실행 취소 / 다시 실행
- 캔버스 지우기
- `Esc`로 종료

## 캔버스와 캡처 중 선택

정적 스냅샷과 내보내기에는 영역 캡처를 사용하세요. 현재 데스크톱 컨텍스트 위에서 실시간 설명을 할 때는 스크린 캔버스를 사용하세요.
