## ปุ่มลัดเริ่มต้น

| การดำเนินการ | ค่าเริ่มต้น |
|---|---|
| การจับภาพเฉพาะส่วน | `F2` |
| สกรีนแคนวาส | `Ctrl+F2 / Cmd+F2` |
| วาง | `F3` |
| ปักหมุดด่วน | `Shift+F2` |

## การดำเนินการที่กำหนดค่าได้เพิ่มเติม (ไม่มีค่าเริ่มต้น)

- Pin from Image
- Pin History
- Toggle Hide/Show All Pins
- Record Screen (เฉพาะ macOS/Windows)

## หมวดหมู่ปุ่มลัด

ปุ่มลัดจัดการตามหมวดหมู่ใน Settings:

- Capture
- Canvas
- Clipboard
- Pin
- Recording (เฉพาะ macOS/Windows; ซ่อนใน Linux beta)

## กลยุทธ์การแก้ไขการชนกัน

หากคีย์ที่กำหนดขัดแย้งกับแอปอื่น:

1. เปลี่ยนการกำหนดการดำเนินการใน Settings > Hotkeys
2. ทดสอบทริกเกอร์ทันที
3. เก็บการดำเนินการที่ใช้บ่อยไว้บนคีย์คอมโบเดียว

## คำแนะนำสำหรับทีม

สำหรับสภาพแวดล้อมที่ใช้ร่วมกัน กำหนดโปรไฟล์ปุ่มลัดมาตรฐานและบันทึกไว้ภายใน
