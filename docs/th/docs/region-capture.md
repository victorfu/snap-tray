## เริ่มจับภาพ

กด `F2` (ค่าเริ่มต้น) เพื่อเข้าสู่การจับภาพเฉพาะส่วน เคอร์เซอร์จะเปลี่ยนเป็นรูปกากบาท และตัวช่วยเคอร์เซอร์จะปรากฏตามการตั้งค่า `Cursor companion` ใน Settings

## พฤติกรรมการเลือก

### การเล็งที่แม่นยำ

- โหมดแว่นขยายให้ข้อมูลป้อนกลับระดับพิกเซล
- ความกว้างและความสูงจะแสดงใกล้กับส่วนที่เลือก
- กดค้าง `Shift` เพื่อล็อกอัตราส่วนภาพ

### การเลือกที่รับรู้หน้าต่าง

เมื่อการตรวจจับหน้าต่างพร้อมใช้งาน SnapTray สามารถตรวจจับหน้าต่างที่อยู่ใต้เคอร์เซอร์ได้ เพื่อให้คุณเลือกได้ด้วยคลิกเดียว

### ยกเลิกและลองใหม่

คลิกขวาหรือกด `Esc` เพื่อยกเลิกการเลือกปัจจุบัน

## เวิร์กโฟลว์หลายส่วน

กด `M` เพื่อสลับโหมดการจับภาพหลายส่วน คุณสามารถรวบรวมหลายส่วนแล้วส่งออกเป็นเอาต์พุตรวมหรือแยกกันก็ได้

## การดำเนินการหลังเลือก

หลังจากเลือกพื้นที่เสร็จแล้ว แถบเครื่องมือรองรับ:

- เครื่องมือมาร์กอัปและแก้ไข (Arrow/Pencil/Marker/Shape/Text/Mosaic/Step Badge/Emoji/Eraser)
- `OCR` (เฉพาะ macOS/Windows; ซ่อนใน Linux beta) และ `QR Code Scan`
- `Auto Blur` (ใบหน้า + ข้อมูลสำคัญ)
- `Pin`, `Save` และ `Copy`

## ปุ่มลัดที่มีประโยชน์

| การดำเนินการ | ปุ่มลัด |
|---|---|
| เริ่มการจับภาพเฉพาะส่วน | `F2` |
| คัดลอกรหัสสีใต้เคอร์เซอร์ (เมื่อโหมดแว่นขยายเปิดอยู่) | `C` |
| ล็อกอัตราส่วน | กดค้าง `Shift` |
| สลับหลายส่วน | `M` |
| ยกเลิก | `Esc` |

## ขั้นตอนถัดไป

ไปที่ [เครื่องมือมาร์กอัป](/th/docs/annotation-tools/) เพื่อแก้ไขภาพที่จับก่อนบันทึก/คัดลอก
สำหรับการบันทึกแบบเต็มหน้าจอบน macOS หรือ Windows ใช้ [การบันทึกหน้าจอ](/th/docs/recording/)
