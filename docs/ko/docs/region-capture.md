## 캡처 시작

`F2`(기본값)를 눌러 영역 캡처를 시작합니다. 커서가 십자선으로 바뀌고, 설정의 `커서 보조 도구` 설정에 따라 커서 컴패니언이 나타납니다.

## 선택 동작

### 정밀 타기팅

- 돋보기 모드로 픽셀 수준의 피드백 제공
- 선택 영역 근처에 너비와 높이 표시
- `Shift`를 누른 채 드래그하면 종횡비 고정

### 창 인식 선택

창 감지를 사용할 수 있는 경우, SnapTray는 커서 아래의 창을 감지하여 한 번의 클릭으로 선택할 수 있습니다.

### 취소 및 재시도

우클릭하거나 `Esc`를 눌러 현재 선택을 취소합니다.

## 다중 영역 워크플로

`M`을 눌러 다중 영역 캡처 모드를 전환합니다. 여러 영역을 수집한 후 합쳐서 또는 개별적으로 내보낼 수 있습니다.

## 선택 후 동작

선택이 완료되면 툴바에서 다음 동작을 사용할 수 있습니다.

- 주석 + 편집 도구 (화살표/연필/마커/도형/텍스트/모자이크/단계 배지/이모지/지우개)
- `OCR` (macOS/Windows 전용; Linux beta에서는 숨김) 및 `QR 코드 스캔`
- `자동 블러` (얼굴 + 자격 증명)
- `URL 공유` (선택된 결과를 업로드하고 공유 링크 반환)
- `고정`, `저장`, `복사`

## 유용한 단축키

| 동작 | 단축키 |
|---|---|
| 영역 캡처 시작 | `F2` |
| 커서 아래 색상 코드 복사 (돋보기 모드 활성 시) | `C` |
| 종횡비 고정 | `Shift` 누르기 |
| 다중 영역 전환 | `M` |
| 취소 | `Esc` |
