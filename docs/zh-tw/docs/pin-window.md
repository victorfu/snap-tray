## 建立釘選視窗

可由截圖結果按 `Enter`，或從托盤選單「Pin from Image...」建立。

## 核心控制

### 滑鼠操作

- 拖曳移動
- 滾輪縮放
- `Ctrl` + 滾輪調透明度
- 邊緣拖曳調整大小

### 鍵盤操作

| 動作 | 按鍵 |
|---|---|
| 順時針旋轉 | `1` |
| 逆時針旋轉 | `2` |
| 水平翻轉 | `3` |
| 垂直翻轉 | `4` |
| 關閉 | `Esc` |

## 右鍵選單功能

可用 Copy、Save、OCR、QR Scan、Live Update、Click-through 與 Close。

## 釘選視窗內標註

點選鉛筆圖示可展開釘選專用標註工具列，快速加註不需回到主截圖流程。
