การบันทึกหน้าจอใช้ได้เฉพาะบน macOS และ Windows เท่านั้น Linux beta ไม่รวมการบันทึก และ UI การบันทึกรวมถึงรายการ CLI help ถูกซ่อนไว้

## จุดเริ่มต้นการบันทึก

- เมนูถาดระบบ: Record Screen
- CLI: `snaptray record start [--screen N]`

## วงจรชีวิตการบันทึก

1. เลือกหน้าจอที่ต้องการบันทึกเมื่อมีการแสดงผลหลายจอ
2. การบันทึกเริ่มทันทีบนหน้าจอที่เลือก
3. ใช้แถบควบคุมแบบลอยเพื่อตรวจสอบระยะเวลาและหยุดการบันทึก
4. คลิก Stop เพื่อส่งออก

## รูปแบบเอาต์พุต

| รูปแบบ | การใช้งานทั่วไป |
|---|---|
| MP4 (H.264) | บทช่วยสอน การบันทึกยาว |
| GIF | สาธิตวนซ้ำสั้น |
| WebP | คลิปเคลื่อนไหวขนาดเบา |

## ตัวเลือกเสียง

การจับเสียงใช้ได้สำหรับการบันทึก MP4 เมื่อแพลตฟอร์มและแหล่งสัญญาณปัจจุบันรองรับ:

- ไมโครโฟน
- เสียงระบบ
- การจับแบบผสม

GIF และ WebP ส่งออกแบบไม่มีเสียง

## การปรับคุณภาพ

เปิด Settings > Recording แล้วปรับเฟรมเรต คุณภาพ การนับถอยหลัง และพฤติกรรมการแสดงตัวอย่าง
