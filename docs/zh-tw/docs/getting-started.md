## 開始前準備

- macOS 14+ 或 Windows 10+
- 已授權 Screen Recording 權限
- 建議授權 Accessibility 權限（視窗偵測用）

## 首次啟動檢查

### 1. 確認托盤圖示

啟動後確認系統托盤有 SnapTray，選單應包含 Region Capture、Screen Canvas 與 Settings。

### 2. 測試第一張截圖

按 `F2` 拖曳選取區域，確認選取時會顯示放大鏡與尺寸資訊。

### 3. 測試儲存與複製

在截圖工具列中：

- `Ctrl/Cmd + C` 複製
- `Ctrl/Cmd + S` 儲存

### 4. 確認輸出路徑

開啟 Settings > Files，檢查截圖與錄影輸出資料夾。

## 建議先調整的項目

- 檔名格式
- 工具列樣式
- 熱鍵衝突（若 `F2` 與其他軟體重疊）
