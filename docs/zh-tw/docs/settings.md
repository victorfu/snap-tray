## 設定分頁

### General

開機啟動、顯示語言、App 主題與 CLI 安裝/移除。

macOS 另外提供 Screen Recording / Accessibility 權限狀態與一鍵開啟系統設定捷徑。

### Hotkeys

依功能分類集中管理快捷鍵。

### Advanced

區域擷取放大鏡開關、截圖快捷提示、Auto Blur 強度/類型、釘選視窗預設值（透明度、透明度步進、縮放步進、快取上限），以及 debug build 才會出現的 MCP 開關。

### Watermark

浮水印圖片、是否套用到截圖/錄影、透明度、縮放、邊距與位置預覽。

### OCR

OCR 語言優先序與辨識後行為。

### Recording

幀率、預設輸出格式、預覽優先的匯出流程、音訊來源/裝置與倒數計時。

### Files

截圖/錄影輸出路徑與檔名格式模板。

### Updates

目前版本、自動更新檢查頻率、上次檢查狀態與立即檢查。

### About

App 名稱、版本、版權、作者與網站連結。

## 推薦基準配置

- 一般教學使用 30 FPS MP4
- 截圖與錄影分開資料夾管理
- 每天自動檢查更新
