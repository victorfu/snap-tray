## เข้าสู่โหมดแคนวาส

กด `Ctrl+F2 / Cmd+F2` (ค่าเริ่มต้น) หรือใช้เมนูถาดระบบ "Screen Canvas"

## เครื่องมือที่ใช้ได้

- ดินสอ / มาร์กเกอร์ / ลูกศร / รูปทรง
- ข้อความ / แบดจ์ขั้นตอน / อีโมจิ
- เลเซอร์พอยน์เตอร์สำหรับการนำเสนอ

## โหมดพื้นหลัง

### กระดานไวท์บอร์ด

เหมาะสำหรับการฉายในห้องมืดหรือภาพโน้ตที่สะอาด

### กระดานดำ

คอนทราสต์ดีกว่าเมื่อเน้นสีสว่าง

## การควบคุมเซสชัน

- Undo / Redo
- ล้างแคนวาส
- `Esc` เพื่อออก

## เมื่อไหรควรใช้แคนวาสแทนการจับภาพ

ใช้การจับภาพเฉพาะส่วนสำหรับสแนปช็อตแบบนิ่งและการส่งออก ใช้สกรีนแคนวาสสำหรับการอธิบายสดบนบริบทเดสก์ท็อปปัจจุบัน
