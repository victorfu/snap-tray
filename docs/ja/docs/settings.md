## 設定タブ

OCR と録画タブは macOS と Windows でのみ利用可能です。Linux beta では
これらの機能が含まれていないため、両タブは非表示になっています。

### 一般

起動時に起動、表示言語、アプリテーマ、CLI のインストール/アンインストール。

macOS では、このタブに画面収録とアクセシビリティの許可状態が表示され、ワンクリックでシステム設定へのショートカットを使用できます。

### ホットキー

機能カテゴリ別の集中管理ホットキーエディター。

### 詳細設定

範囲キャプチャのルーペ切り替え、キャプチャショートカットのヒント、ぼかしの強度/種類、ピン留めウィンドウのデフォルト（透明度、透明度ステップ、ズームステップ、キャッシュ制限）。

### 透かし

透かし画像の選択、スクリーンショット/録画への適用有無、透明度、スケール、マージン、位置のプレビュー。

### OCR

OCR の言語順序と認識後の動作。

### 録画

フレームレート、デフォルト出力フォーマット、プレビュー優先エクスポート動作、音声ソース/デバイス、カウントダウン。

### ファイル

スクリーンショットと録画のフォルダー、およびファイル名フォーマットのテンプレート。

### アップデート

現在のバージョン、自動確認間隔、最終確認状態、手動で今すぐ確認。

### バージョン情報

アプリ名、バージョン、著作権、作者、ウェブサイトリンク。

## 推奨ベースラインプロファイル

- macOS/Windows では標準チュートリアル用に 30 FPS MP4
- スクリーンショット用（利用可能な場合は録画用）の専用フォルダー
- 毎日の自動アップデート確認を有効化
