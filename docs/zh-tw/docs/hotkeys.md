## 預設快捷鍵

| 動作 | 預設 |
|---|---|
| Region Capture | `F2` |
| Screen Canvas | `Ctrl+F2` |
| Paste | `F3` |
| Quick Pin | `Shift+F2` |

## 其他可設定動作（預設未綁定）

- Pin from Image
- Pin History
- Toggle Hide/Show All Pins
- Record Full Screen

## 分類管理

Settings > Hotkeys 會依功能分類管理：

- Capture
- Canvas
- Clipboard
- Pin
- Recording

## 衝突處理建議

1. 在 Hotkeys 內重新綁定。
2. 立即測試是否生效。
3. 常用功能優先使用單段組合鍵。

## 團隊建議

團隊環境可制定統一熱鍵配置，降低教學與支援成本。
