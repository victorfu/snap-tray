## 錄影入口

- 托盤選單：Record Full Screen
- 截圖工具列：按 `R`

## 錄影生命週期

1. 選擇區域或全螢幕目標。
2. 按 Start 開始。
3. 透過浮動控制列 Pause/Resume。
4. 按 Stop 匯出檔案。

## 輸出格式

| 格式 | 適用情境 |
|---|---|
| MP4 (H.264) | 長時教學、產品 Demo |
| GIF | 短循環示意 |
| WebP | 輕量動態片段 |

## 音訊選項

依平台支援可選：

- 麥克風
- 系統音訊
- 混合錄音

## 品質調校

在 Settings > Recording 可調整 FPS、畫質、倒數計時與預覽行為。
