## 設定分頁

### General

開機啟動、工具列樣式、釘選快取與 CLI 安裝/移除。

### Hotkeys

依功能分類集中管理快捷鍵。

### Recording

幀率、格式、畫質、倒數、預覽與音訊來源。

### Files

截圖/錄影輸出路徑與檔名格式模板。

### OCR / Watermark / Updates

語言偏好、浮水印位置與自動更新檢查頻率。

## 推薦基準配置

- 一般教學使用 30 FPS MP4
- 截圖與錄影分開資料夾管理
- 每週自動檢查更新
