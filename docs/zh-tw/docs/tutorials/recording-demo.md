## 目標

產出一段讓利害關係人快速看懂的短錄影。

## 建議長度與格式

- 建議長度：20-90 秒
- MP4：一般團隊最通用
- GIF/WebP：適合貼在聊天或文件

## 操作流程

### 1. 定義錄影範圍

- 全螢幕：完整導覽
- 區域：聚焦示範

可從托盤選單啟動，或在截圖選區按 `R` 開始。

### 2. 開錄前先設定

在 Settings > Recording 確認：

- 幀率與畫質
- 倒數秒數
- 音訊來源（麥克風/系統音）

### 3. 有節奏地錄製

1. 開始錄影。
2. 滑鼠移動放慢，避免觀眾迷失。
3. 切換場景前可先暫停。
4. 證明重點後立刻停止。

### 4. 依受眾輸出

- MP4：工單、簡報、正式回報
- GIF/WebP：日常同步、快速預覽

建議檔名：

`onboarding-flow-checkout-2026-02-24.mp4`

### 5. 補一段脈絡文字

描述三件事：

- 觀眾該看哪個重點
- 預期結果
- 實際結果（若是 bug 回報）

## 品質檢查清單

- 影片一開始就是目標狀態，沒有多餘準備畫面。
- 滑鼠與操作節奏易於理解。
- 輸出格式符合投遞場景。
- 檔名與文字說明足夠清楚。
