## デフォルトのホットキー

| アクション | デフォルト |
|---|---|
| 範囲キャプチャ | `F2` |
| スクリーンキャンバス | `Ctrl+F2 / Cmd+F2` |
| 貼り付け | `F3` |
| クイックピン | `Shift+F2` |

## 追加設定可能なアクション（デフォルトなし）

- 画像からピン留め
- ピン履歴
- すべてのピンの表示/非表示を切り替え
- 画面を録画（macOS/Windows のみ）

## ホットキーカテゴリ

ホットキーは設定内でカテゴリ別に管理されます：

- キャプチャ
- キャンバス
- クリップボード
- ピン
- 録画（macOS/Windows のみ；Linux beta では非表示）

## 競合時の対応

キーバインドが別のアプリと競合する場合：

1. 設定 > ホットキー でアクションを再バインドします。
2. すぐにトリガーをテストします。
3. よく使うアクションは単一キーコンボに保ちます。

## チームへの推奨

共有環境では、標準ホットキープロファイルを定義して内部で文書化してください。
