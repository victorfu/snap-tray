## 10 分鐘你會完成什麼

- 權限設定完成
- 第一張截圖與標註
- 第一個釘選視窗
- 第一段錄影輸出
- 快捷鍵衝突確認

## 前置條件

- macOS 14+ 或 Windows 10+
- Screen Recording 權限
- 建議開啟 Accessibility 權限（視窗偵測與流程操作更順）

## 步驟 1：確認托盤執行狀態（1 分鐘）

1. 啟動 SnapTray。
2. 確認系統托盤有圖示。
3. 打開選單並確認有：
   - Region Capture
   - Screen Canvas
   - Settings

若托盤未出現，請先看[疑難排解](/zh-tw/docs/troubleshooting/)。

## 步驟 2：測試截圖流程（2 分鐘）

1. 按 `F2` 截圖。
2. 拖曳任一區域，確認有放大鏡與尺寸資訊。
3. 加一個箭頭標註。
4. 按 `Ctrl/Cmd + C`，貼到任一文字編輯器。
5. 按 `Ctrl/Cmd + S`，確認檔案有成功輸出。

## 步驟 3：測試釘選流程（2 分鐘）

1. 再按一次 `F2` 截圖。
2. 按 `Enter` 將畫面釘選。
3. 測試縮放、移動。
4. 切換 App，確認釘選仍可見。

## 步驟 4：測試錄影流程（3 分鐘）

1. 從托盤啟動錄影，或在選區按 `R`。
2. 錄 5-10 秒。
3. 停止並輸出 MP4。
4. 在 Settings > Files 確認輸出路徑。

## 步驟 5：處理快捷鍵衝突（2 分鐘）

開啟 Settings > Hotkeys，調整衝突按鍵。

最低建議先確認這四個：

- Region Capture
- Screen Canvas
- Quick Pin
- Paste
