## 進入畫布模式

按預設 `Ctrl+F2`，或從托盤選單啟動 Screen Canvas。

## 可用工具

- Pencil / Marker / Arrow / Shape
- Text / Step Badge / Emoji
- Laser Pointer（簡報模式）

## 背景模式

### Whiteboard

適合投影、白底簡報與教學註記。

### Blackboard

適合需要高對比亮色標記的情境。

## 會話控制

- Undo / Redo
- Clear
- `Esc` 離開

## 何時用畫布、何時用截圖

Region Capture 適合靜態輸出；Screen Canvas 適合即時講解與演示。
