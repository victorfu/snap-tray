## 啟動截圖

按預設熱鍵 `F2` 進入 Region Capture，游標會切換為十字準星並顯示放大鏡。

## 選取行為

### 精準定位

- 放大鏡提供像素級預覽
- 選取框旁會顯示寬高尺寸
- 按住 `Shift` 可鎖定比例

### 視窗感知選取

當平台支援視窗偵測時，可自動辨識游標下視窗並一鍵選取。

### 取消重選

右鍵或 `Esc` 可取消當前選取。

## 多區域流程

按 `M` 切換多區域擷取。你可累積多個區塊，再選擇合併輸出或分開儲存。

## 常用快捷鍵

| 動作 | 快捷鍵 |
|---|---|
| 啟動區域截圖 | `F2` |
| 鎖定比例 | 按住 `Shift` |
| 切換多區域 | `M` |
| 由當前區域錄影 | `R` |
| 取消 | `Esc` |
