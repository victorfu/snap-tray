## キャプチャの開始

`F2`（デフォルト）を押して範囲キャプチャを開始します。カーソルがクロスヘアに切り替わり、設定の「カーソルコンパニオン」設定に従ってカーソルコンパニオンが表示されます。

## 選択の動作

### 精密な位置指定

- ルーペモードでピクセルレベルのフィードバックを提供
- 選択範囲の近くに幅と高さを表示
- `Shift` を押しながらドラッグするとアスペクト比を固定

### ウィンドウを認識した選択

ウィンドウ検出が利用可能な場合、SnapTray はカーソル下のウィンドウを検出し、ワンクリックで選択できます。

### キャンセルと再試行

右クリックまたは `Esc` を押して現在の選択をキャンセルします。

## マルチリージョンワークフロー

`M` を押してマルチリージョンキャプチャモードを切り替えます。複数の範囲を収集し、結合または個別の出力としてエクスポートできます。

## 選択後のアクション

選択が完了すると、ツールバーは以下をサポートします：

- 注釈・編集ツール（矢印/ペン/マーカー/図形/テキスト/モザイク/ステップバッジ/絵文字/消しゴム）
- `OCR`（macOS/Windows のみ；Linux beta では非表示）および `QR コードスキャン`
- `自動ぼかし`（顔と認証情報）
- `共有 URL`（選択した結果をアップロードして共有リンクを返す）
- `ピン留め`、`保存`、`コピー`

## 便利なショートカット

| アクション | ショートカット |
|---|---|
| 範囲キャプチャを開始 | `F2` |
| カーソル下の色コードをコピー（ルーペモード有効時） | `C` |
| アスペクト比を固定 | `Shift` を押しながら |
| マルチリージョンの切り替え | `M` |
| キャンセル | `Esc` |

## 次のステップ

保存・コピー前にキャプチャを編集するには [注釈ツール](/ja/docs/annotation-tools/) へ進んでください。
macOS または Windows でのフルスクリーン録画には [録画](/ja/docs/recording/) を使用してください。
