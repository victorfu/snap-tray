## 建立釘選視窗

可由截圖結果按 `Enter`，或從托盤選單「Pin from Image...」建立。

## 核心控制

### 滑鼠操作

- 拖曳移動
- 滾輪縮放
- `Ctrl` + 滾輪調透明度
- 邊緣拖曳調整大小

### 鍵盤操作

| 動作 | 按鍵 |
|---|---|
| 順時針旋轉 | `1` |
| 逆時針旋轉 | `2` |
| 水平翻轉 | `3` |
| 垂直翻轉 | `4` |
| 關閉 | `Esc` |

## 右鍵選單功能

右鍵選單提供：

- 視圖與版面：Show Toolbar、Show Border、Move to Screen、Merge Pins、Adjust Region Layout
- 影像控制：縮放預設、影像處理（旋轉/翻轉/Crop/透明度步進）、平滑顯示
- 輸出流程：Copy、Save、Open Cache Folder、Beautify、Watermark、Live Update
- 互動與關閉：Click-through、Close、Close All Pins

## 釘選視窗內標註

點選鉛筆圖示可展開釘選專用標註工具列，快速加註不需回到主截圖流程。

- 繪圖工具：Pencil、Marker、Arrow、Shape、Text、Mosaic、Step Badge、Emoji、Eraser
- 實用工具：Crop、Measure
- 動作工具：OCR、QR Code Scan、Share URL、Save、Copy
