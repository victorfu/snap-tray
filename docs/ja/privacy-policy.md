## 概要

SnapTray はローカル優先のスクリーンショット・注釈アプリです。録画機能は macOS と Windows のみ対応しています。Linux beta はスクリーンショットと注釈のみサポートします。ネットワーク接続は、共有 URL のアップロードを明示的に実行するとき、または任意のアップデート確認が動作するときに限られます。プライバシーの保護と透明性のある運用を重視しています。

**要点：** SnapTray はローカル優先で動作し、テレメトリ・追跡・分析は一切ありません。ネットワーク通信は、共有 URL（ユーザー起動の画像アップロード）とアップデート確認（バージョン文字列のみ）の 2 つの明示的な機能に限定されます。

## データ収集

**データは一切収集しません。** 具体的には：

- テレメトリや利用状況の分析は行いません
- クラッシュレポートを外部サーバーに送信しません
- ユーザーの追跡や固有識別子の作成は行いません

## データの保存

SnapTray が使用・生成するすべてのデータはお使いのデバイス上にローカル保存されます：

| データの種類 | 保存場所 | 目的 |
|-------------|----------|------|
| スクリーンショット | ユーザー指定フォルダ（既定：ピクチャ） | キャプチャした画像の保存 |
| 録画 | ユーザー指定フォルダ（既定：ダウンロード） | macOS と Windows の画面録画を保存。Linux beta は録画機能を含みません |
| 設定 | OS の設定ストレージ* | 各種設定の保存 |

*設定の保存場所：
- **macOS:** `~/Library/Preferences/cc.snaptray.plist`
- **Windows:** `HKEY_CURRENT_USER\Software\SnapTray`
- **Linux beta:** Qt/Linux ユーザー設定ストレージ（通常 `~/.config/<organization>/<application>.conf`）
- 新バージョンへのアップグレード後、初回起動時に旧バージョンの設定が自動移行され、移行成功後に旧データは削除されます。

## 権限

SnapTray は機能に応じてシステム権限を要求します：

### macOS

| 権限 | 用途 |
|------|------|
| Screen Recording | スクリーンショットと録画のコア機能 |
| Accessibility | スマートキャプチャ機能のためのウィンドウ検出 |

### Windows

| 権限 | 用途 |
|------|------|
| Full Trust | DXGI 画面キャプチャ、WASAPI 音声キャプチャ、OCR 機能に必要 |

**任意の権限：**
- マイクへのアクセスは、音声録音を選択した場合にのみ要求されます。

## ネットワーク通信

SnapTray が接続するのは以下の 2 つの明示的な機能のみです：

### 共有 URL

共有 URL 機能を明示的に使用すると、キャプチャした画像が `x.snaptray.cc` にアップロードされます。パスワードを任意で設定できます。共有リンクは自動的に有効期限切れになります。ユーザーが操作しない限り、アップロードは一切行われません。

### アップデート確認

SnapTray は新しいリリースを `api.github.com` で確認します。この確認はデフォルトで有効になっており、設定から無効にできます。送信されるのはアプリのバージョン文字列のみです。個人情報・利用状況・デバイス識別子は一切送信されません。ストアインストール（Microsoft Store、Mac App Store 等）の場合はアップデート確認をスキップします。

### SnapTray が行わないこと

- 追跡 SDK や分析ツールを含みません
- Cookie や類似技術を使用しません
- テレメトリやクラッシュレポートを送信しません

- [GitHub リポジトリ](https://github.com/victorfu/snap-tray)

## まとめ

| 質問 | 回答 |
|------|------|
| SnapTray はデータを収集しますか？ | いいえ |
| SnapTray はサーバーにデータを送信しますか？ | 共有 URL を使用するときと、アップデート確認が有効な場合（バージョン文字列のみ）のみ |
| SnapTray は利用状況を追跡しますか？ | いいえ |
| SnapTray は分析ツールを含みますか？ | いいえ |
| SnapTray は自動で外部接続しますか？ | いいえ。アップデート確認はオプトアウト可能で、バージョン番号のみ送信します |
| ファイルはどこに保存されますか？ | すべてお使いのデバイス上にローカル保存されます |
| これらの主張を検証できますか？ | はい、ソースコードは公開されています |
